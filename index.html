Macaron & Canvas: A Paleta de Cores que Você Pode Saborear.

Briefing: Desenvolvi um visual web que integra arte e gastronomia, utilizando uma paleta de cores inspirada em confeitos e aquarelas. A estilização CSS garantirá que o elemento gráfico vertical se posicione com destaque, criando uma interface única e charmosa para um estúdio criativo.
